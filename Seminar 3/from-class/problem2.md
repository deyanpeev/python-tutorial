# Letters contained in a string

Given a list of letters and a string, check if all the list's elements are contained in the string

---
Sample input:
``` python
['a', 'b', 'c']
'some random string'
```
Sample output:
```
False
```
---
Sample input:
``` python
['s', 'r', 't']
'some random string'
```
Sample output:
```
True
