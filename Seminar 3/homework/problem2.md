# Draw a rhombus

Given an integer number, print a rhombus using the char <i>*</i> with as many lines as the given integer. 

Verify that the minimum value of N is 3.

---
Sample input:
``` python
3
```
Sample output:
```
 *
***
 *
```
---
Sample input:
``` python
4
```
Sample output:
```
 *
***
***
 *
```
---
Sample input:
``` python
5
```
Sample output:
```
  *
 ***
*****
 ***
  *
```
