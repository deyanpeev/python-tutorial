# Draw a tree

Given an integer number, print a christmas tree using the char <i>*</i> with as many lines as the given integer. 

Verify that the minimum value of N is 4.

---
Sample input:
``` python
4
```
Sample output:
```
  *
 ***
*****
  *
```
---
Sample input:
``` python
5
```
Sample output:
```
   *
  ***
 *****
*******
   *
```
