# Remove vowels from string list

Given a string of lists remove all vowels in all strings. Vowels are: <i>a, e, i, o, u</i>

---
Sample input:
``` python
["vowel", "consonant"]
```
Sample output:
```
vwl, cnsnnt
```
---
Sample input:
``` python
["Random", "WORDS", "HERE"]
```
Sample output:
```
Rndm, WRDS, HR
