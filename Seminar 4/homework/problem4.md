# Find common elements

Given a list of  lists, print the common elements in all lists

The list string is preset in the program and not passed as input.

---
Sample input:
``` python
[[2, 17, 3, 4, 17], [2, 3, 1, 2], [82, 2, 4, 3, 20, 60 , 4, 4]]
```
Sample output:
```
2, 3
```
---
Sample input:
``` python
[['a', 'b', 'c'], ['b', 'c', 'd'], ['c', 'd', 'e']]
```
Sample output:
```
c
```
